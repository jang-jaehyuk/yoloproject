# 프로젝트 소개
yolo를 활용한 맹견, 입마개 detection을 데이터청년캠퍼스에 참여중인 팀원간의 협업을 진행하기 위한 project입니다.
# 기본 플로우
1. 각자 저장소를 복제
```
$ git clone https://github.com/jang-jaehyuk/yoloproject.git
```
2. 각자의 환경에서 작업
3. 작업물 push
